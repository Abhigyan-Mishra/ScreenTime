# Usage
Track how much you use your computer in a day.

## Features
* Time spent on computer today
* Number of unlocks

## Installation
Download this project and run the following commands to install.

```shell
make
sudo make install
```

## Usage
Run the following command to see your computer usage:

```shell
sudo usage_reader
```
